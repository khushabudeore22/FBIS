Course Name

Web Development with HTML, CSS, JavaScript, Bootstrap and GitHub

subject details

Web development builds and maintains websites/apps, split into Front-End (user-facing: HTML, CSS, JavaScript, design) and Back-End (server/database logic), or Full-Stack (both). Core skills involve HTML for structure, CSS for styling, and JavaScript for interactivity, with modern tools like React, Node.js, Git, VS Code, and frameworks like Bootstrap/Tailwind speeding up development

html

HTML (HyperText Markup Language) is the standard markup language used to create web pages. It defines the structure and content of a webpage using elements like headings, paragraphs, links, images, and more. HTML works alongside CSS (for styling) and JavaScript (for interactivity) to build modern websites.

JavaScript

JavaScript (JS) is a versatile, high-level programming language and a core technology of the World Wide Web, alongside HTML and CSS. It is used primarily to create dynamic and interactive content for websites but has since expanded to enable server-side development, mobile apps, and more.

bootstrap

Bootstrap is a popular, free, open-source front-end framework for building responsive, mobile-first websites quickly, offering pre-written HTML, CSS, and JavaScript for components (like buttons, navbars) and layouts (like the 12-column grid system), simplifying development by providing consistent styling and interactive features without needing complex custom coding.

css

Cascading Style Sheets (CSS) is a style sheet language used to define the presentation and layout of a web page written in a markup language like HTML. It works alongside HTML (which provides structure) and JavaScript (which adds interactivity) to create visually appealing and responsive websites.
